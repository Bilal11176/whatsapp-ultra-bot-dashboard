
# Ultra Bot Dashboard

Dashboard for controlling WhatsApp Ultra Bot

**Owner:** +93795430668

## How to use

1. Open https://github.com/Bilal11176/whatsapp-ultra-bot-dashboard.git
2. Enter your bot server IP: `http://ANDROID_IP:3000`
3. Control everything from the dashboard

## Features
- View captured view-once media
- Send messages to any contact
- Broadcast to all contacts
- Harvest contacts
- Toggle invisible mode
- Live message feed
